About Lumen

Lumen is a true search platform designed to help you find information quickly and efficiently. Its mission is to provide a seamless search experience that connects users with the most relevant content across the web.

Vision behind this project

I envision a world where information is accessible to everyone and is true, precise and with ability to easily proofread from multiple sources. Lumen aims to bridge the gap between ALL data sources and everyday users by providing an intuitive interface and powerful search capabilities.

How It Works

Lumen connects to nodes that search content from various sources. When you perform a search, Lumen queries these nodes and aggregates the results to provide you with the most relevant information.

Open Source

Lumen is built on open-source technologies and is itself open source. I believe in the power of community-driven development and welcome contributions from developers around the world. What you are currently seeing is the frontend built with simple HTML, CSS and JS. You can find its source code on GitHub.
